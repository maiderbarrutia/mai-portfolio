# Maider Barrutia | Personal portfolio

<details>
<summary><b>Español</b></summary>
<br>

Aplicación web bilingüe desarrollada para la presentación de proyectos y perfil profesional. Este proyecto demuestra la integración completa entre una **interfaz de usuario** reactiva y accesible, y una arquitectura modular para la optimización de rendimiento y SEO.

🔗 **Sitio en producción:** [maiderbarrutia.com](https://maiderbarrutia.com)


### 🛠️ Tecnologías utilizadas

* **Frontend:** Next.js (App Router), React, TypeScript, Sass (SCSS)
* **Validación y formularios:** Zod, Web3Forms
* **Despliegue:** Vercel

### ✨ Funcionalidades

* 🌐 **Web bilingüe:** Soporte completo para Español e Inglés con rutas limpias y organizadas.
* 🌙 **Modo oscuro:** Cambiar entre tema claro y oscuro de forma fluida.
* 📱 **Diseño responsive:** Totalmente optimizado para móviles, tablets y ordenadores (estilo Bento Grid).
* ♿ **Accesibilidad & SEO:** Código semántico para que sea accesible para todos y optimizado para buscadores.
* 📬 **Formulario de contacto:** Validación en tiempo real para recibir mensajes de forma segura.

### 📁 Estructura

* `app/` → Páginas y rutas de la aplicación (versión en español e inglés).
* `components/` → Bloques visuales reutilizables (Header, Hero, Proyectos, Experiencia, Contacto).
* `styles/` → Estilos globales y componentes de diseño utilizando Sass.
* `data/` → Información sobre mis proyectos y tecnologías de forma ordenada.

### 📁 Project structure

* `app/` → Application pages and routing (both Spanish and English versions).
